The directory you found this README in is the main file of Michael Hawkins's first year MSc project 'Unveiling hidden targets'.

File names: below is a summary of what each folder in this directory contains.

bin: A temp file used to deposit all new files made from the scripts in src.
src: A file containing all scripts involved in this project. A separate README is included in the src directory containing information relating to the use of those scripts.
output; contains endpoint files generated from the scripts.
Raw_data: Contains all data required for the use of these scripts.
report: Contains all report related materials.
README.txt: this txt document.
